This is a sleek, modern, and highly secure web-based password generator application. It allows users to create strong, random passwords by customizing length and including specific character sets (uppercase, lowercase, numbers, and symbols). The application features real-time **password strength feedback** and one-click copy functionality.

## ‚ú® Features

* **Custom Length:** Generate passwords from 4 up to 32 characters.
* **Character Customization:** Toggle inclusion of Uppercase (A-Z), Lowercase (a-z), Numbers (0-9), and Symbols (!@#$%).
* **Guaranteed Complexity:** The generator ensures at least one character from every selected set is included, maximizing security.
* **Real-time Strength Indicator:** Provides immediate visual feedback (Weak, Medium, Strong, Excellent) based on complexity and length.
* **One-Click Copy:** Seamlessly copy the generated password to the clipboard.
* **Sleek Design:** Features a modern, dark-themed UI built with pure HTML, CSS, and JavaScript.
